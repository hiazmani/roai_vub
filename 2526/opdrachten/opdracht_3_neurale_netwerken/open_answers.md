# Open Answers

## Reflection — Rashevsky and modern neural networks

Acceptable answers should mention at least three similarities and three differences. Students do not need to use exactly this wording.

### Possible similarities

- Both use nonlinear activation rather than purely linear response.
- Both can include recurrent or feedback connections between units.
- Both can show threshold-like behavior.
- Both can be described mathematically as systems of interconnected units.
- Both can display stable and unstable states depending on parameters.

### Possible differences

- Rashevsky's model is hand-designed and not trained from data.
- Modern neural networks usually learn parameters from examples; Rashevsky's model does not.
- Rashevsky studies a very small dynamical circuit, not a large multilayer architecture.
- Modern neural networks are often used for prediction or classification; Rashevsky's model is used to explain a psychological/biophysical phenomenon.
- Modern deep learning relies on optimization procedures such as gradient-based learning; Rashevsky's model does not.
- Rashevsky focuses on trajectories, equilibria, hysteresis, and qualitative dynamics rather than task performance on datasets.

### What would count as a strong answer

A strong answer should make clear that Rashevsky's model is "neural-network-like" in structure, but very different from modern machine learning in purpose, scale, and training method.
